\doxysubsection{Detailed Description}
3D visualization system for autonomous driving simulation 

Handles Open\+GL rendering pipeline, vehicle model display, and multiple view modes. Integrates with Qt framework for \mbox{\hyperlink{classGUI}{G\+UI}} interaction.
