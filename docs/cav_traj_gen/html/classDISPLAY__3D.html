Detailed Description

3D visualization system for autonomous driving simulation

Handles OpenGL rendering pipeline, vehicle model display, and multiple view modes. Integrates with Qt framework for GUI interaction.

Maintains historical path buffer.

  • Stores recent trajectory points from optimal path
  • Keeps maximum 500 points (≈25 seconds at 20Hz)
  • Filters points closer than 0.2m
  • Clears buffer when jumps >3m detected
